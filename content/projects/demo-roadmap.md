这是一个项目文档风格的页面，展示 Quartz 也适合写轻量产品说明或工程笔记。

## 下一步建议

1. 把你的 Markdown 笔记复制到 `content/`。
2. 在 `quartz.config.ts` 里改站点标题、主题颜色和正式域名。
3. 运行 `npx quartz build` 生成 `public/` 静态文件。
4. 按官方 Hosting 文档部署到 GitHub Pages、Cloudflare Pages、Vercel 或 Netlify。
